## 🚀 快速开始

### 第一步：了解 Seedance 2.0

Seedance 2.0 是字节跳动的 AI 视频生成模型，核心特点：

- **有智能** - 具备导演思维，自动编排分镜
- **有知识** - 自带世界知识，无需百科全书式描述
- **全能参考** - 支持图片(≤9张)、视频(≤3个)、音频(≤3个)同时参考

### 第四步：编写提示词

按照模板格式填写你的内容：

```
【风格】_____风格，_____秒，_____比例，_____氛围

【时间轴】
0-X秒：[镜头] + [画面] + [动作] + [特效]
X-Y秒：[镜头] + [画面] + [动作] + [特效]
...

【声音】_____配乐 + _____音效 + _____对白

【参考】@图片1 _____，@视频1 _____
```

### 第五步：上传素材并生成

1. 上传参考素材（图片/视频/音频）
2. 在提示词中使用 `@素材名` 引用
3. 选择时长和画幅
4. 点击生成

## 📚 核心概念

### "写意图、不写细节"

在有世界知识的场景下，只需描述意图，让 AI 发挥导演思维：

```
✅ 推荐：生成一个精美高级的兰州拉面广告，注意分镜编排
❌ 不推荐：描述如何揉面、拉面、浇汤的每个步骤细节
```

### 全能参考语法

```
@图片1 作为首帧
@图片2 作为尾帧
@图片3 作为角色形象参考
@视频1 参考运镜方式
@音频1 用于配乐
```

## 💡 使用技巧

### 一句话生成

满足以下条件时，一句话即可生成高质量视频：

- ✅ 有世界知识（餐饮品牌/健身动作/历史时期等）
- ✅ 不需要特定角色一致性
- ✅ 不需要特殊运镜风格

示例：
- "生产一个精美高级的兰州拉面广告，注意分镜编排"
- "帮我生成一个讲述无印良品这个品牌的宣传片"

### 版权规避

使用知名角色可能被拒绝，解决方案：

- ❌ 不要直接写角色名字
- ✅ 使用 "Figure 1"、"Figure 2" 替代
- ✅ 描述外观特征

## ⚠️ 重要限制

| 项目 | 限制 |
|------|------|
| 真人脸部素材 | ❌ 不支持 |
| 文件总数 | 最多 12 个（图片≤9 + 视频≤3 + 音频≤3） |
| 视频总时长 | ≤15 秒 |
| 音频格式 | 仅支持 MP3 |
| 生成分辨率 | 目前 720P 为主 |
